## dev-os 工具 · 通用坑（用本 OS 时常见,先记着）
- **zsh 数组**：`for f in $STR` 在 zsh 不按词分 → 用数组 `ARR=(...)`。
- **种坏门必抓要验"门真会红"**：写完闸门检查,先种一个已知违规跑一遍、确认它 FAIL/WARN,否则门可能是纸做的（如退化矩阵让变异测试假绿）。
- **派生量(计数/索引)别手敲**：从源 token 脚本派生(`build_*`),并设 validate 核对;源 token 名要规范统一,一漂就连锁错。
